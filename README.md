# pdd-185 物流追踪地图

物流追踪数据可视化大屏，展示包裹在抽象地图上的实时位置和运输状态。

## 功能

- 抽象 SVG 地图，支持缩放、拖拽
- 包裹标记按状态着色（蓝/绿/红）
- 左侧包裹列表，支持按状态筛选
- 顶部统计栏：发货量、在途、已签收、异常
- 点击标记弹出详情卡片
- 数据每 3 秒自动刷新模拟位置移动
- 支持普通/卫星地图模式切换

## 技术栈

- React 18 + TypeScript + Vite
- TailwindCSS
- ECharts（预留）
- Lucide React 图标

## 运行

```bash
cd pdd-185
npm install
npm run dev
```

默认端口 3000。
